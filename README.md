# AI Weboldalak — Next.js 14 + Framer Motion

Minimal Next.js 14 (app router) TypeScript projekt, Framer Motion animációval.

## Parancsok

```bash
npm install
npm run dev   # fejlesztői szerver
npm run build # build
npm start     # production
```

## Deploy Vercelre

```bash
npx vercel
npx vercel deploy --prod
```
